# Student Event Registration System

A robust, console-based backend application built using Core Java to automate club event registrations and optimize student data management.

## 🚀 Key Features
- **Object-Oriented Architecture:** Developed using clean OOP principles (Encapsulation, Polymorphism, and Data Abstraction).
- **Dynamic Data Management:** Leveraged Java Collections Framework (`ArrayList`) for real-time participant tracking and manipulation.
- **Capacity Constraint Validation:** Implemented structural boundary conditions to automatically restrict over-booking once venue limits are reached.

## 🛠️ Tech Stack & Concepts
- **Language:** Java (JDK 8 or higher)
- **Concepts:** Collections Framework, Conditional Logic, User Input Validation, Console I/O.

## 💻 How to Run the Application
1. Clone the repository to your local machine.
2. Navigate to the source folder and compile the code:
   ```bash
   javac src/EventSystem.java
   ```
3. Execute the program:
   ```bash
   java src/EventSystem
   ```
